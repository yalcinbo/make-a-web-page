Important Concepts

Intro to Programming Lesson 01

How it Works

The web is a network of computers that communicate with one another via a HTTP request to a server. Once the request has been made the server sends the HTML document to the computer that made the request, this is when the computer interprets the HTML document via a web browser and displays it on the users screen.

HTML

HTML stands for Hypertext Markup Language, which form the majority of content on the Internet. HTML consists of text content to determine what you see and markup content to determine how it looks. CSS is used to modify the visual aspect of the text to a greater extent

HTML Vocabulary

Tag = An HTML tag is always contained within angled brackets tags are used to tell browsers the type of each element.
Element = An element is everything within a set of opening and closing tags, HTML documents are made up of these elements.
Attribute = This is a property of a HTML element.

Types of Elements

HTML elements are either inline or block. Block elements form an invisibile box arount the content, whereas inline elements do not.

Important to Remember

In programming it is important not to forget that computers always translate the code literally, thus there may be simple syntax errors that cause frustration. It is important check for accuracy and keep calm when working on code!

Intro to Programming Lesson 02

Developer Tools

The Developer Tools tab within the browser allows you to view the page as it is from a syntax stand point, you can also see the styling of the page and the details of all the content/images.

Document Object Model

This is also reffered to as the DOM or the tree, due the branches that it has when drawn out. In a browser this structure shows up as a series of nested boxes. There are boxes inside of boxes, and so on...

Text Editors

When writing code is best to use specialized text editors that will make it easier to write and read the code. It is possible to use notepad or other word documents, but it will be incredibly difficult to write substantial code with that approach.

Intro to Programming Lesson 03

Avoiding Repetition

Avoiding Errors: When programmers don't avoid repetition, they will often have to do the same thing over and over. This may mean copy and pasting certain style attributes into many HTML elements or rewriting the same code (sometimes with minor differences) many times. This can lead to errors when the programmer decides to make a change to something. If they don't dilligently make that same change everywhere the repeated code appears, problems will arise.

CSS

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.
